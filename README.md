# GenHisDoc

GenHisDoc is a layout recognition dataset aiming to train generalistic models for manuscript and printed documents. GenHisDoc was created by reusing annotations froms previous datasets work : 

|Name       |               Description |      Corpus |   Licence | 
|S-VED | The Spherae Sacrobosco dataset |Printed Books| CC BY 4.0 |
|HoraeLS\_V2|The Book of hours dataset  |Manuscripts  | CC BY 4.0 |
